## Project
The project is a Wi-Fi enabled smart medication dispenser that uses IoT Blockchain technology. The goal is to create a system capable of managing several dispensers throughout a healthcare facility with the goal of tracking usage and prevent tampering. 

Additional details are available in the project wiki
